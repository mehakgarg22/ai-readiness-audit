# 🤖 AI Readiness Audit Tool

### The Project
This tool was developed as a solution for an internship technical assessment. Its primary objective is to evaluate how efficiently a website can be parsed and understood by AI-driven search engines (e.g., Perplexity, SearchGPT).

### Key Features
* **Automated Audit Simulation:** Runs a technical check on a provided URL.
* **Inference Scoring:** Generates a readiness score (0-100) based on machine-readability benchmarks.
* **Technical Insights:** Highlights critical missing elements like JSON-LD Schema, robots.txt AI permissions, and semantic HTML hierarchy.

### Tech Stack
* **Frontend:** HTML5, CSS3
* **Logic:** Vanilla JavaScript (ES6+)
